TROUBLESHOOTING:
----------------
Q: Backend won't start?
A: Check if Python is installed: python --version

Q: Port 5001 already in use?
A: Close other Python processes

Q: No results in browser?
A: 
   1. Make sure backend window is open
   2. Check browser console (F12)
   3. Try: http://127.0.0.1:5001/api/search/all
      (Should show JSON data)
