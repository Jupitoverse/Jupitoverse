# Phase 1 Presentation — Data Annotation Platform (demo folder)

All presentation deliverables in one place.

## Four deliverables

| # | Deliverable | File / How to get it |
|---|-------------|----------------------|
| 1 | **MD file** (presentation content in Markdown) | `Phase1_Presentation.md` |
| 2 | **PPT generated by assistant** | Same content as the Python-generated deck; use `Phase1_Presentation.md` as source. |
| 3 | **PPT generated via Python** | Run the script below. Output: `Phase1_Data_Annotation_Platform.pptx` |
| 4 | **Prompt for Gamma** | `GAMMA_PROMPT.md` — copy the prompt into [Gamma](https://gamma.app) to generate a similar deck. |

## Generate the .pptx (Python)

From this **demo** folder:

```bash
pip install python-pptx
python generate_phase1_presentation.py
```

Or on Windows: double-click `run_generate_ppt.bat`.

Output: `Phase1_Data_Annotation_Platform.pptx` (created in this folder).

## Files in this folder

- `Phase1_Presentation.md` — Full deck in Markdown (slides 1–12).
- `Phase1_Presentation_Content.txt` — Plain-text version of slide content.
- `generate_phase1_presentation.py` — Script that creates the .pptx.
- `GAMMA_PROMPT.md` — Copy-paste prompt for Gamma.app.
- `requirements.txt` — Python dependency (python-pptx).
- `run_generate_ppt.bat` — Windows: run to install deps and generate PPT.
- `Phase1_Data_Annotation_Platform.pptx` — Generated after running the script above.
